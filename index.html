IoT - Internet of Things

Our project

IoT is a collective term for technologies of a global infrastructure of information societies.
We will describe how the IoT can be used in daily life.

Positive aspects

We are going to describe useful and good things of IoT.
Some examples might be:

  • Using IoT products in industrial environments
  • Applications for smart home products

Negative aspects

In this section we are going to describe the negative impact it has on people's daily lives.
